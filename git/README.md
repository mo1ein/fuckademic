## Git
### some most used git commands

revert unstaged changes(modified):
```
git checkout -- <file>
```

revert staged changes(after git add):
```
git reset <file>
```
rollback the last commit:
```
git revert HEAD
```

show the commit log in one line:
```
git log --oneline
```

show the commit log as a graph:
```
git log --graph
```
